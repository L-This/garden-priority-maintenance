# نظام مؤشر أولوية صيانة الحدائق - نسخة بوابات المشاريع المستقرة

هذه نسخة منظفة ومستقرة قبل إضافة الهيكل الإداري.

## الصفحات
- `/dashboard` لوحة النتائج وبوابات المشاريع
- `/assessment` صفحة تقييم حديقة جديدة

## تحتوي على
- بوابة كافة المشاريع في المنتصف
- بوابات المشاريع تحتها
- إخفاء المؤشرات حتى اختيار بوابة
- عرض بيانات الحدائق بعد اختيار مؤشر
- تفاصيل المعايير والصور
- Supabase للبيانات الحقيقية
- بيانات تجريبية لا تحفظ في القاعدة
- حذف تقييم حقيقي
- تصدير CSV
